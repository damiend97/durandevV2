Design



The first step of the process always starts with getting the creative juices flowing. This is when I usually use Figma or even a good ole' piece of paper to sketch out the layout of the website. In this phase I also like to find typographies that will go well with the theme of the website. This is also a good time for me to create any graphics and hunt down any images that will be used in the site.

Develop



The next phase is to turn the idea into a reality. 99% of this phase is slamming code into a text editor like Visual Studio. There is a lot of back and fourth between code and web browsers. Now is when I would make sure the site is mobile responsive and also Search Engine Optimized.

Deploy



The last step obviously is deployment. Now that the website is built, it needs to be connected to the internet so everyone can make use of it. This includes setting a domain (the web address for your site), hosting (where your domain and website live), and SSL (the security measures for the site).
